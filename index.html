Features

하나의 플랫폼에서
AI부터 로봇 제어까지

데이터 수집, AI 학습, 판단, 로봇 행동까지 — 피지컬 AI의 전체 파이프라인을 단일 환경에서 경험합니다.

66종 블록

프리모블록 코딩

PLC 자동화 개념을 블록으로 추상화. 66종 카드로 IF/LOOP/WHILE, GPIO, MQTT, AI 분류까지. PLC 스캔 사이클과 1:1 매칭되는 순차 배치 구조.

94 노드

프리모노드 ML 캔버스

노드 기반 비주얼 ML 파이프라인. KNN/SVM/결정트리 등 12종 알고리즘 + PCA/t-SNE + 혼동행렬/ROC 시각화. 학습된 모델을 프리모블록에서 바로 활용.

5종 알고리즘

AI 비전 스튜디오

5단계 Step Wizard로 데이터 수집 → 학습 → 평가 → 추론. KNN/DNN/SVM/결정트리/로지스틱회귀 + Grad-CAM 해석 + 노드 캔버스 연동.

110 FB

ST 코드 에디터

IEC 61131-3 표준 47개 + 고급 22개 + 프리모스튜디오 확장 41개 = 110개 함수 블록. CODESYS/TIA Portal SCL 내보내기로 실제 PLC에 바로 적용.

SVG 차트 7종

데이터 분석 & 리포트

히스토그램/히트맵/산점도/ROC 등 7종 SVG 차트. 학습 세션 16종 이벤트 기록 + AI 분석 리포트 + PDF 내보내기.

6레벨

6단계 학습 시스템

입문(초등) → 기초 → 중급 → 고급 → 심화(고등) → 실무. 각 레벨에서 모드/기능을 자동 제어. 교사가 학습 단계를 선택하면 UI가 맞춰 변환.

Learning Path

초등학생부터 산업 현장까지

같은 플랫폼에서 6단계를 올리며 성장합니다. 프리모블록에서 시작해 ML 파이프라인을 설계하는 전문가로.

1

프리모블록

초등 고학년

PLC 자동화를 블록으로 추상화. 66종 카드로 순차/반복/조건 프로그래밍 기초를 실물 장비와 함께 학습합니다.

2

ST 코드

중학생

IEC 61131-3 표준 ST 언어로 텍스트 코딩. 카드↔코드 양방향 변환. CODESYS/TIA Portal 내보내기.

3

AI 비전

고등학생

5단계 Wizard로 AI 모델 학습. 5종 알고리즘 + Grad-CAM 해석. 학습된 모델을 하드웨어에 적용.

4

프리모노드

대학 / 산업

94개 노드로 ML 파이프라인 설계. KNN부터 k-Means까지. 모델 학습 → 평가 → 하드웨어 연동을 하나의 캔버스에서.

Requirements

시스템 요구사항

시뮬레이션 모드로 하드웨어 없이도 전체 기능을 체험할 수 있습니다.

💻 소프트웨어

  • macOS 12 Monterey 이상 (Apple Silicon / Intel)
  • Windows 10 / 11 (64-bit)
  • 디스크 공간 500MB 이상
  • RAM 4GB 이상 (8GB 권장)

🔧 하드웨어 키트 (선택)

  • 6축 협동 로봇팔
  • 컨베이어 벨트 (스텝모터)
  • AI 카메라 (Wi-Fi 무선)
  • 적외선 감지 센서

🌐 연결 방식

  • 로봇팔: LAN 케이블 직결
  • AI 카메라: Wi-Fi 무선 연결
  • 컨베이어: USB 케이블 연결
  • 인터넷 연결 불필요 (오프라인 사용 가능)
